# Portfolio | Marc Schüttforth

Modern one-page portfolio built with Next.js, React, and Tailwind CSS.

## 🚀 Tech Stack

- Next.js 16.2.4 & React 19.2.4
- Tailwind CSS 4
- Responsive & Mobile-first

## 📦 Sections

- Hero Section
- Tech Stack & Skills
- Project Showcase (3 Projects)
- About Me
- Contact & CV Download

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📝 Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run start` - Run production
- `npm run lint` - Linting
